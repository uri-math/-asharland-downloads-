  
The files in this folders are Singular (W. Decker, G.-M. Greuel, G. Pfister and H. Schonemann, [https://www.singular.uni-kl.de](https://www.singular.uni-kl.de/)) codes for the examples shown the article  
  
**Ayse Sharland and Jacob Smith, \"Fitting Ideals without a Presentation\", to appear in the Journal of Algebra and Computation (2025).**  
**(<https://www.worldscientific.com/worldscinet/ijac>)**  
  
  
To run the codes, copy the files to a directory in \$PATH, and use the following line within Singular:  
  
execute(read(\"Example31elim\"));  
  
(Use read(\"Example31elim\"); to have Singular print the contents of the file.)  
  
Alternatively, open the files in any of the commonly used text editors, and copy and paste into Singular directly.  
  
  
**Please cite the article above, if sharing the codes in any kind of platform or projects.**
